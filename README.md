 # Sales Dashboard – Internship Project

## Project Overview
This project presents an interactive dashboard built in Excel to analyze sales performance. The dashboard highlights revenue trends by region, product category, and time period, helping to turn raw data into meaningful business insights.

## Features
- Raw Data sheet with structured sales records
- Pivot Table summarizing totals by region and category
- Dashboard with:
  - Slicers for Region, Product, and Date
  - KPI box showing Total Revenue
  - Charts (Bar, Pie, Line) for different perspectives
  - Insights section summarizing key findings

## Goal
To demonstrate how data can be transformed into clear visual reports that support decision‑making.

## Skills Applied
- Data organization and cleaning
- PivotTable creation
- Dashboard design with slicers and KPIs
- Extracting business insights from data
